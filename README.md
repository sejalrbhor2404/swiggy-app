# Swiggy Clone - 3 Tier DevOps Project 🚀

## 📌 Project Overview

This project is a 3-tier Swiggy-like food ordering web application deployed on AWS using DevOps tools.

---

## 🏗️ Architecture

User → Load Balancer → ECS (Docker Container) → RDS (MySQL)

---

## ⚙️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js (Express)
* Database: AWS RDS (MySQL)
* Containerization: Docker
* CI/CD: Jenkins
* Infrastructure: Terraform
* Configuration Management: Ansible
* Cloud: AWS (ECS, ECR, ALB)

---

## 🚀 Features

* Food menu display
* Add to cart functionality
* Backend API integration
* Database connectivity (RDS)
* Dockerized application
* CI/CD pipeline using Jenkins
* Infrastructure provisioning using Terraform

---

## 🔧 Deployment Steps

1. Build Docker Image
2. Push to AWS ECR
3. Deploy on ECS
4. Connect to RDS
5. Configure Load Balancer
6. Setup CI/CD using Jenkins
